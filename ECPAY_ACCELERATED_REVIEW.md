# 綠界加速審核完整指南

本文件提供綠界商家帳號加速審核所需的完整條款範本和檢查清單。

## 🎯 目標

透過提供完整的網站條款和資訊，將綠界審核時間從 **1-3 個工作天**縮短至 **當天或隔天**。

## 📋 必備文件清單

### 3. 退換貨政策（Refund Policy）

**檔案位置建議**：`/refund` 或 `/refund-policy`

**必須包含的章節**：

```markdown
# 退換貨政策

## 一、適用範圍
本政策適用於 Premium 訂閱服務。

## 二、退款條件

### 數位訂閱服務特殊說明
本服務為數位訂閱服務，一經開通即開始提供服務。

### 退款期限
- **7 天內**：可申請全額退款
- **超過 7 天**：按比例退款（扣除已使用期間）

### 退款計算方式
退款金額 = 已付款金額 × (剩餘天數 / 總訂閱天數)

範例：
- 年繳方案（NT$1,200），使用 30 天後申請退款
- 退款金額 = 1,200 × (335 / 365) = NT$1,100

## 三、退款流程
1. 透過 Email 或客服系統申請退款
2. 提供訂單編號和退款原因
3. 我們將在 3-5 個工作天內處理
4. 退款將退回原付款方式

## 四、不適用退款情況
1. 已使用超過訂閱期間的 50%
2. 違反服務條款被終止服務
3. 重複申請退款

## 五、取消訂閱
1. 用戶可隨時取消訂閱
2. 取消後不會自動續約
3. 已付款期間的服務仍可使用至到期日
4. 取消後不會自動退款（需依退款政策申請）

## 六、聯絡方式
退款相關問題，請聯絡：
- Email: [您的客服信箱]
- 處理時間：3-5 個工作天
```

## ✅ 最終檢查清單

在提交綠界審核前，請確認：

### 文件完整性
- [ ] 服務條款已建立（至少 500 字）
- [ ] 隱私權政策已建立（至少 500 字）
- [ ] 退換貨政策已建立（至少 300 字）
- [ ] 聯絡資訊頁面已建立

### 網站整合
- [ ] Footer 有所有條款連結
- [ ] 註冊/登入頁面有條款連結
- [ ] 付款頁面有條款連結
- [ ] 所有連結可正常存取（不需要登入）

### 內容正確性
- [ ] 聯絡資訊真實有效
- [ ] 地址完整（含郵遞區號）
- [ ] Email 可正常接收信件
- [ ] 電話號碼格式正確（如有）

### 網站功能
- [ ] 網站可正常訪問
- [ ] 沒有明顯錯誤
- [ ] 付款流程正常（測試環境）
- [ ] 響應式設計正常

## 🚀 提交審核

完成以上檢查後：

1. **登入綠界商家後台**
2. **上傳所需文件**（身分證、存摺等）
3. **填寫網站資訊**（包含所有條款連結）
4. **提交審核**

## 💡 加速審核額外建議

1. **完整填寫商家資料**：不要留空白欄位
2. **提供真實資訊**：綠界會驗證聯絡資訊
3. **網站功能完整**：確保基本功能正常運作
4. **及時回應**：如有問題，及時回覆綠界客服
5. **準備齊全**：一次提交完整資料，避免補件延遲
